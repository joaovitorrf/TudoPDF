Como converter Word para PDF no celular (Android e iPhone)

📅 Abril 2025⏱️ 3 min de leitura🏷️ Conversão
Início › Blog › Como converter Word para PDF no celular (Android e iPhone)

Precisa converter um arquivo Word (.docx) para PDF e está só com o celular? Sem problema. Existem várias formas de fazer isso gratuitamente, sem precisar instalar aplicativos pagos.

Método 1 — Pelo Google Docs (sem instalar nada)

É o método mais simples e funciona em Android e iPhone:

  1. Abra o Google Drive no celular e faça upload do arquivo .docx
  2. Toque no arquivo para abrir no Google Docs
  3. Toque nos três pontos (⋮) no canto superior direito
  4. Selecione "Compartilhar e exportar"
  5. Toque em "Salvar como" e escolha PDF
  6. Baixe o arquivo PDF gerado
💡 Dica

O Google Docs preserva muito bem a formatação de documentos simples. Para documentos com formatação complexa (múltiplas colunas, caixas de texto especiais), o resultado pode variar.

Método 2 — Microsoft Word no celular

Se você tem o app Word instalado (gratuito para arquivos simples):

  1. Abra o arquivo .docx no Word
  2. Toque no ícone de menu (três linhas)
  3. Vá em "Exportar"
  4. Escolha "PDF"
  5. Salve no dispositivo ou compartilhe diretamente

Método 3 — Pelo WPS Office (gratuito)

O WPS Office é uma alternativa gratuita ao Microsoft Office com função nativa de exportar para PDF:

  1. Instale o WPS Office (grátis na Play Store ou App Store)
  2. Abra o arquivo .docx
  3. Toque em "Ferramentas" → "Exportar para PDF"

iPhone vs Android — tem diferença?

O processo é muito parecido em ambos. No iPhone, você também pode usar o Arquivos para abrir documentos Word e, com alguns aplicativos de terceiros, exportar para PDF. O Google Docs funciona igual nos dois sistemas.

Conclusão

Converter Word para PDF pelo celular é totalmente possível e gratuito. O Google Docs é a opção mais prática por não exigir nenhuma instalação adicional. Para comprimir o PDF resultante ou fazer outras edições, use as ferramentas gratuitas do TudoPDF.
