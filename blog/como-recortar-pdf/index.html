Como recortar (cortar margens) de um PDF online

📅 Junho 2025 ⏱️ 4 min de leitura 🏷️ Edição
Início › Blog › Como recortar PDF

PDFs gerados por escâneres muitas vezes têm margens enormes e desnecessárias. Apresentações exportadas como PDF às vezes ficam com espaço em branco ao redor do conteúdo. A ferramenta de recorte resolve isso: você ajusta exatamente qual área quer mostrar em cada página.

O que é "recortar" um PDF?

Recortar (ou "crop") um PDF significa definir uma área visível menor do que a página original. O conteúdo fora da área recortada fica oculto — mas não é apagado permanentemente. Tecnicamente, o PDF continua armazenando o conteúdo completo; o que muda é o quanto é exibido ao leitor.

⚠️ Atenção

Por isso, recortar um PDF não é uma forma segura de esconder informações confidenciais. Use a ferramenta Ocultar/Redigir PDF para remover conteúdo de forma permanente.

Como recortar pelo TudoPDF

  1. Acesse www.tudopdf.site/tools/recortar-pdf/
  2. Faça o upload do seu PDF
  3. Ajuste as margens (topo, base, esquerda, direita) para cada página ou para todas de uma vez
  4. Visualize o resultado e baixe o arquivo recortado

Quando recortar um PDF é útil?

  • Escâneres que adicionam bordas brancas enormes ao redor das páginas
  • Livros digitais com margens de impressão desnecessárias para leitura em tela
  • Apresentações com espaço vazio ao redor dos slides
  • Documentos que serão usados em e-readers como Kindle, onde o espaço é pequeno
  • Preparar PDFs para impressão em tamanhos específicos

Recortar afeta o tamanho do arquivo?

Pouco. Como o conteúdo fora da área recortada ainda fica armazenado no arquivo, o tamanho não reduz significativamente. Para reduzir o peso do PDF, use a ferramenta de compressão depois de recortar.

Conclusão

Recortar um PDF é a solução ideal para eliminar margens excessivas e tornar o documento mais apresentável. Leva menos de um minuto pelo TudoPDF.
