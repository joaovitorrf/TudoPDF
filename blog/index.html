✍️ Como adicionar novos posts ao blog

O blog do TudoPDF usa arquivos HTML estáticos hospedados no GitHub. Você pode adicionar posts diretamente pelo GitHub, sem precisar de servidor ou CMS.

  1. Acesse o repositório do projeto no GitHub
  2. Navegue até a pasta blog/
  3. Crie uma nova pasta com o slug do post, ex: blog/como-comprimir-pdf-grande/
  4. Crie o arquivo index.html dentro dessa pasta usando o template abaixo
  5. Faça commit e o Vercel faz deploy automático em segundos
  6. Adicione a URL no sitemap.xml e faça ping ao Google: https://www.google.com/ping?sitemap=https://tudopdf.com.br/sitemap.xml

Dica SEO: O nome da pasta vira a URL. Use palavras-chave reais: como-comprimir-pdf-para-email ranqueia melhor que post-1. Sempre adicione a tag <link rel="canonical"> e o schema BlogPosting para máxima indexação.
