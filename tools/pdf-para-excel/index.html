📈 PDF para EXCEL

Detecta tabelas no PDF automaticamente e gera uma planilha .xlsx editável. Ideal para PDFs com dados tabulares.

⚠️ Limitação importante: a detecção de tabelas é baseada no alinhamento do texto. PDFs com tabelas bem definidas terão ótimos resultados. PDFs escaneados ou com layout complexo podem precisar de ajustes manuais após a conversão.
